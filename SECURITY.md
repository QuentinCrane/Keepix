# 安全政策

## 支持范围

当前主要维护最新主分支和公开 Release。旧版本问题会根据风险和维护成本处理。

## 报告安全问题

请不要在公开 Issue 中贴出：

- 真实照片、视频、精确位置或隐私路径。
- `local.properties`、签名密钥、keystore 密码、环境变量。
- 可直接复用的攻击细节或用户数据样本。

如果仓库已开启 GitHub Security Advisories，请优先使用私有安全报告；否则请通过维护者公开资料中的私密联系方式沟通。普通崩溃、权限提示或 UI 问题可以使用 Issue 模板反馈。

## 安全边界

看了么是本地优先的媒体整理工具。核心原则：

- 不默认上传用户照片或视频。
- 不加入账号、会员、支付、广告或远程授权校验。
- 公共媒体永久删除应走 Android 系统授权与用户确认流程。
- 签名密钥、Release 密码和本地配置只应保存在开发者本机或 CI Secret 中。
