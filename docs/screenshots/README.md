# 截图指南

GitHub 首页最好放真实截图。建议使用隐私安全的测试媒体，不要使用个人照片、视频、精确位置或真实相册路径。

## 推荐截图

1. `home.png`：首页与整理入口。
2. `photo-clean.png`：图片整理流。
3. `video-clean.png`：视频整理流。
4. `favorites-trash.png`：收藏与回收站。
5. `settings.png`：设置中心。
6. `tablet.png`：平板或横屏适配。
7. `dark-mode.png`：深色模式。

## 图片要求

- 优先使用 PNG。
- 手机截图建议宽度 1080px 左右。
- README 中展示的图片可以压缩到 1200px 以下，避免仓库体积过大。
- 如果图片里出现真实媒体内容，请先替换、打码或使用测试素材。
